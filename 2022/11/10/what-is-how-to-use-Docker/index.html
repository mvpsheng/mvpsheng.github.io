what is & how to use Docker

what is Docker

Docker 是一个用来开发、发布和运行应用的开放平台。
docker 工具目的是让开发者更简单的应用容器来创建、部署和运行应用。

why Docker

Docker 可以让基础架构和应用程序分离，这样可以快速交付软件。
Docker可以像管理应用软件一样管理基础架构。
利用Docker的快速交付、测试和部署方法，可以显著减少写代码和生产环境运行之间的延迟。
